## Webmotors test

### Versions

```bash
$ node -v #12.16.x
$ npm -v #6.14.x
```

### Run project

```bash
$ yarn
$ yarn start
```

### Available scripts

- `start` - Starts project
- `build` - Generate production build
- `test` - Run tests
- `test:watch` - Run test with --watch
- `lint` - Run lint with eslint
- `lint:fix` - Fix lint errors
- `lint:css` - Run lint with stylelint
